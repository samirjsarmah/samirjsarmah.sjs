## Overview

DevTrack is a collaborative project management tool that brings real-time Kanban boards, sprint planning, and team collaboration into a single, polished interface.

## Features

- Drag-and-drop Kanban boards with optimistic updates
- Real-time collaboration via Supabase Realtime
- Sprint planning and velocity tracking
- GitHub integration for automatic PR status cards
- Slack notifications for task updates
